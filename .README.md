# teerapap's dotfiles

## Installation

	cd ~ && git clone git://github.com/teerapap/dotfiles.git .

## Set solarzied dark theme to gnome-terminal for GNOME (>=3.8, using dconf)

	.bin/gnome-solarized-dark.sh [GNOME-PROFILE-ID]
